What is .then() used for, and what is .catch() used for?

The then callback is used when the promise is resolved. One can also chain then method callbacks. Each then receives the previous result of the last then's return value. The catch callback is executed when the promise is rejected. Typically errors are sent to the catch.

What are good use cases for Promises?

Promises are typically used when using the fetch() API. Promises are a fantastic tool for working with asynchronous code.

What other libraries/functions can you find that uses Promises?

Promises can be used both in the browser and Node.js. One can npm install promises and then require them for Node.js. This library also includes some extensions including nodeify and denodeify. jQuery also has a .promise() method that returns a dynamically generated Promise that is resolved once all actions of a certain type have ended.
